## Quick Setup

1. Add `nuxt-locokit` dependency to your project

```bash
# Using pnpm
pnpm add -D nuxt-locokit

# Using yarn
yarn add --dev nuxt-locokit

# Using npm
npm install --save-dev nuxt-locokit
```

2. Add `nuxt-locokit` to the `modules` section of `nuxt.config.ts`

```js
export default defineNuxtConfig({
  modules: [
    'nuxt-locokit'
  ]
})
```

That's it! You can now use Nuxt Module in your Nuxt app âœ¨

## Development

```bash
# Install dependencies
npm install

# Generate type stubs
npm run nuxt:prepare

# Build the playground
npm run nuxt:build

# Develop with the playground
npm run dev

# Build module
npm run build

# Run ESLint
npm run lint

# Run Vitest
npm run test
npm run test:watch

# Release new version
npm run release
```
